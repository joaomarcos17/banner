Home Network VS Enterprise Network


Both share the same principles of ip addressing , subnet masks , DHCP.
The implement is different (Enterprise Network) use fulltime managers to keep things working.
Enterprise router, switches setting up these equipments is more envolved.
The (Home Network) is comprised of routers, switches, wifi.
The (Enterprise Network) they usually separate pieces of equipments and often far from each other.
In a home gateway the routers have just two interfaces setting up is in a graphic user interface (GUI).
In (Enterprise Network) might have two or more interfaces giving on ip number on one these routers is usually done In (CLI) command line interface and envolve several steps.
In (Enterprise Network) is the extensive use of subnetworking to create manageable groups of users in large organisations you could have subnetworks for these groups of staff.
